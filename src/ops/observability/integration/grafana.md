# Grafana 接入

本文档介绍如何将 Aries 系统接入 Grafana 可视化平台。

## 系统要求

1. 软件要求
   - Grafana 9.x
   - 数据源支持
   - 插件支持
   - Web 服务器

2. 硬件要求
   - CPU：2核以上
   - 内存：4GB以上
   - 存储：20GB以上
   - 网络：千兆网卡
